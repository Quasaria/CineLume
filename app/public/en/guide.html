1. Getting started

Just open quasaria.github.io/CineLume. No sign-up, no email, no password. The app is immediately usable.

On first launch you see the theatrical release calendar for the current week in France. You can change the country via the Filters button at the top.

2. Finding releases

The central navigation bar shows the current month and year. You can:

  • Tap the month to open a picker (12-month grid + year arrows)
  • Use left/right arrows to navigate month by month
  • Click a week (W1, W2, W3...) to select it
  • Click the calendar icon to return to today

The Filters button opens a panel to refine by genre, streaming platform (Netflix, Prime Video, Disney+, etc.), release type (theatrical, streaming, festival) and max runtime.

You can also search any movie, actor or director via the magnifier in the top right. Recent search history (last 8) is shown when the input is empty.

3. Favorites and watchlist

On each movie card, two buttons at the top right:

  • Heart (♥) = add to Favorites. Favorites trigger release notifications.
  • Bookmark = add to Watchlist (to watch later).

To view your lists, click the corresponding icons in the navbar. Both tabs live in the same "My movies" modal — switch between them without closing.

Movies are automatically grouped: This week, Upcoming, Released, Unknown date.

4. Custom lists

For thematic collections (Date night, Tarantino marathon, To show my parents...), use Lists:

1
Burger menu → My lists
2
New list button, give it a name
3
Click the emoji icon left of the name to pick an emoji (🎬 🍿 ⭐ ❤️...)
4
From a movie modal, Add to a list button
5
In the expanded list, drag the GripVertical handle at the bottom-left of a poster to reorder
6
Share button (link icon) generates a URL you can send. Recipient opens it, list auto-imports.

5. Release notifications

To be notified on the day your favorite movies release:

  1. Add movies to your Favorites (♥)
  2. Settings → Notifications section → Enable
  3. Allow in the browser popup
  4. (Optional) Configure quiet hours (e.g. 10pm-8am) to avoid being disturbed at night

Notifications work in the open browser and in installed PWA mode. No third-party push service: everything is local (browser Notification API).

6. Swipe and Pick modes

When you can't decide what to watch, two alternative modes are in the burger menu:

Pick a movie

A modal that randomly picks a movie from the chosen source (Watchlist / Favorites / All my lists). Optional genre filter. "Another one" button to reroll, "View details" to open the full modal.

Swipe selector mode

Tinder-style: choose a source (This week / Watchlist / Favorites). Card by card:

  • Swipe right or heart button = add to watchlist
  • Swipe left or X button = skip
  • Undo button brings back the last card if you mis-swiped
  • On desktop: left/right keyboard arrows also work
  • Scroll down within a card to see synopsis, genres, director

7. Letterboxd import

If you already use Letterboxd and want to migrate your watchlist:

1
On Letterboxd: Settings → Import & Export → download your CSV
2
In CineLume: Settings → Letterboxd section → Import
3
Select the CSV file. Progress shows in real time with current title.
4
A detailed report lists skipped films (ambiguous titles or unknown to TMDB)

You can also export your CineLume watchlist in Letterboxd CSV format via the Export button in the same section, to migrate elsewhere if desired.

8. Install CineLume on your phone

CineLume is a Progressive Web App, installable on all devices:

On iPhone / iPad (Safari)

  1. Open CineLume in Safari
  2. Tap the Share icon (square with arrow)
  3. Select Add to Home Screen

On Android (Chrome)

Chrome automatically suggests an "Install app" button at the bottom of the screen after a few visits, or via menu ⋮ → Install app.

On desktop (Chrome / Edge)

An Install icon appears at the right of the address bar. Otherwise, menu ⋮ → Install CineLume.

Once installed, CineLume launches like a native app (without browser chrome). Long-press the home screen icon for shortcuts: Favorites and Pick a movie.
