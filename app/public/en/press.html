CineLume press kit

One-sentence pitch

CineLume is a free web app and PWA, with no ads, no subscription and no sign-up, that tracks movie and streaming releases week by week.

Extended pitch (~50 words)

CineLume is the agenda for movie and streaming releases. Users browse the week's calendar, filter by country and platform, save films to favorites or watchlist, share lists by link, and receive release notifications. No account required, no data sent to any server — everything is stored locally in the browser.

Key facts

Type
Web app · Progressive Web App (installable on iOS/Android/desktop)
Business model
None — free, no ads, no subscription, no in-app purchase
Sign-up
None — no email, no password, no account
Storage
Browser localStorage only — no CineLume server
Tracking
Zero — no analytics, no third-party cookies, no telemetry
Languages
French, English
License
MIT, open source
Data sources
TMDB (The Movie Database) for movies, JustWatch via TMDB for streaming
Tech stack
React 19, Vite 7, TypeScript, Tailwind, Zustand, TanStack Query, Workbox PWA

Notable features

  • Calendar of theatrical releases, week by week, by country
  • Full-text search of movies + actors + directors
  • Dedicated person page with photo, biography, filmography
  • Unlimited favorites and watchlist, stored locally
  • Custom lists with emoji and link sharing
  • Tinder-style swipe selector mode with undo
  • Random "Pick a movie" mode with filters
  • Notifications with configurable quiet hours
  • Letterboxd CSV import/export
  • Offline mode (Service Worker)
  • Spoiler-free mode (titles and posters blurred)
  • Multilingual FR/EN, light/dark theme
