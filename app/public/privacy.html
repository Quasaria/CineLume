Politique de vie privée

Mis à jour le __BUILD_DATE_HUMAN_FR__

Résumé en une phrase : CineLume n'a pas de serveur, ne collecte aucune donnée personnelle, n'utilise aucun cookie de tracking, aucune télémétrie, aucun analytique tiers. Tout ce que tu fais reste sur ton appareil.

Quelles données sont collectées ?

Aucune. CineLume n'a pas de serveur ni de base de données. Tes favoris, ta watchlist, tes listes personnalisées, tes préférences (thème, langue, filtres, notifications), ton historique de recherche sont stockés exclusivement dans le stockage local de ton navigateur (localStorage).

Quelles requêtes externes l'application fait ?

  • api.themoviedb.org — pour récupérer les données des films (titre, synopsis, casting, posters, bandes-annonces). Géré par TMDB selon leur politique.
  • image.tmdb.org — CDN TMDB pour les posters et backdrops.
  • www.youtube.com — uniquement quand tu cliques pour lire une bande-annonce. Géré par YouTube selon la politique Google.

Aucune autre requête externe. Aucune vers un serveur CineLume (il n'y en a pas).

Cookies et tracking

  • Aucun cookie n'est posé par CineLume.
  • Aucun pixel publicitaire (Meta, Google Ads, etc.).
  • Aucun analytique tiers (Google Analytics, Mixpanel, Plausible, etc.).
  • Aucune télémétrie applicative.
  • Aucun fingerprinting du navigateur.

Notifications push

Si tu actives les notifications dans Paramètres, CineLume utilise l'API Notification du navigateur, locale. Aucun service push tiers n'est utilisé. Les notifications sont déclenchées par le code de l'app lui-même quand tu rouvres l'application.

Stockage local

Le stockage local du navigateur (localStorage) contient :

  • Tes favoris et watchlist (titres, IDs TMDB, dates de sortie)
  • Tes listes personnalisées avec leurs films
  • Tes préférences (thème, langue, filtres actifs, mode spoiler-free, heures de calme des notifications)
  • Les recherches récentes (8 dernières)
  • Un compteur de visites (anti-spam pour la banner d'installation PWA)
  • Un cache TMDB local pour ne pas re-fetcher les mêmes films

Tu peux vider tout ce contenu via Paramètres → "Vider le cache" ou en effaçant les données du site dans ton navigateur.

Export et portabilité

Tu peux exporter ta watchlist au format CSV Letterboxd à tout moment via Paramètres → Letterboxd → Exporter. Aucun verrou propriétaire.

Open source

Le code source complet est disponible sur GitHub sous licence MIT. Tu peux vérifier toi-même que ces affirmations sont vraies.

Modifications

Si cette politique est modifiée, la date "Mis à jour" en haut de page sera changée. L'historique des versions est visible dans Git.
